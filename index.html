About Me

안녕하세요.
프론트 엔드 개발자가 되고 싶은 전진희 입니다.

제가 이 분야에 관심을 가지게 된 계기는
2019년 워홀로 오스트리아에서 지내고 있을 때 였습니다.
락다운으로 집에만 머물러야 했던 그 때, 우연히 코딩이라는 것을 알게 되었습니다.
'취미로 좀 공부해볼까' 하는 마음으로 처음 코딩에 관심을 두게 되었습니다. 하다보니 신기하고 어렵지만 배워나가고 막히는 부분을 알게 되었을 때의 보람도 있었습니다.
평소에 배우고 공부하는 것을 좋아하기 때문에 취미가 아니라 직업으로 도전해봐도 좋겠다고 느꼈고 더 공부해서 프론트엔드 개발자가 되어야겠다고 생각했습니다.

약 8개월의 시간동안 혼자 독학을 하며 마크업 언어와 자바스크립트를 배웠고, 전체적인 UI / UX를 구성하는 부분에서 어려움을 느껴 국비지원으로 웹 퍼블리셔 과정을 수강하며 UI / UX를 구상하고 디자인하는 과정을 배웠습니다. 올해 초에는 정보처리 기사를 공부해서 정보처리 기사 자격증을 가지고 있습니다.

웹사이트를 만들 때 필요한 HTML / CSS / JAVASCRIPT를 다룰 줄 알고, 현재 REACT와 NODE.JS를 공부하고 있습니다.
